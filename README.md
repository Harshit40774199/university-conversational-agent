\# University Conversational Agent



\## Overview

This project is a context-aware and explainable university chatbot developed as part of a Masters dissertation project.



The system is designed to support students by providing academic information such as:



\- Coursework deadlines

\- Submission procedures

\- Module details

\- Academic regulations

\- Learning resources

\- Student support services



\---



\## Key Features



\### Conversational AI

\- Rasa framework

\- DIET intent classifier

\- Dialogue management

\- Multi-turn interactions



\### Full-Stack Architecture

\- Frontend: HTML, CSS, JavaScript

\- Backend: Java Spring Boot

\- Database: MySQL

\- REST API integration



\### Advanced Features

\- Context-aware conversations

\- Explainable AI (intent + confidence + source links)

\- Retrieval-supported responses

\- Monitoring and logging (lightweight MLOps)



\---



\## System Architecture



Frontend → Spring Boot Backend → Rasa Engine → MySQL Database
